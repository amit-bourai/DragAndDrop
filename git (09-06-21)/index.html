1.Introduction to Git

What is Git?

Git is a distributed version control system (DVSC)

  • Each user has a local copy of the complete histoey of the project, which is known as a repository.
  • User can work offline.
  • Can easily synchronize repositories.
  • Use github, gitLab, bitbucket accounts.

Git

Git is a free and opean source software project meaning that the code that implements Git is punlicly available. No single company own Git, and anyone can make contribution to improve it.
